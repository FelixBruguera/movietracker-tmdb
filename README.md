# Movie Tracker

A web app for searching, tracking and rating movies and TV Shows

## Live demo

[Check out the live demo](https://movietracker.felixbruguera0.workers.dev/)

## Features

- **Discover and filters**: You can filter movies by release year, average TMDB rating, genre, language, streaming service or keywords
- **Saving searches**: You can save filter combinations
- **Search by name**: Search movies, tv shows or people
- **Media Details**: Cast, crew, production companies, service availability and similar media
- **Diary logs**: Track the movies you've watch
- **Reviews**: Rate movies on a 10-point scale with optional written reviews
- **Lists**: Create public or private collections
- **Watchlists**: Create a list that automatically removes movies when you rate or log them
- **Social features**: Follow or clone other users lists
- **Stats**: Charts and lists with different stats about the user's ratings and viewing habits

## Build with

- **Backend**: Hono
- **Frontend**: React
- **Routing**: React Router
- **Auth**: Better Auth
- **Styling**: Tailwind and Shadcn
- **Database**: Cloudflare D1
- **Testing**: Vitest, Playwright
- **Libraries**: Axios, Drizzle, React Query, Zustand, date-fns, Lucide, Zod, Recharts
